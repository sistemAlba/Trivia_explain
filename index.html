Permiso de ubicación

Este permiso sí requiere autorización del usuario. En esta demo se solicita solo para mostrar cómo una página puede pedir acceso.

Antes de escanear un QR

  1. Verifica quién colocó el QR.
  2. Revisa la dirección del sitio antes de ingresar datos.
  3. Desconfía de premios, urgencia o curiosidad excesiva.
  4. No aceptes permisos si no entiendes para qué se usarán.
Esta demo solo guarda conteos agregados: total, Android, iPhone, otros, permisos permitidos y permisos bloqueados. No guarda ubicación, IP, nombre ni datos personales.
